# 緊急應變通用指引

## 一、任何災難發生時的通用原則
1. **確認自身安全**：先確保自己的安全，再幫助他人。
2. **撥打緊急電話**：有人受傷或受困，立即撥打 119。
3. **通報位置**：向救援人員說明確切地址、樓層、受困人數。
4. **收聽官方資訊**：收聽政府廣播，依指示行動，不信謠言。
5. **保存體力與物資**：災難後初期資源可能短缺，合理分配使用。

## 二、緊急包準備清單
### 每人建議備妥（可維持 72 小時）
- 飲用水：每人每天 3 公升 × 3 天 = 9 公升
- 乾糧：不需烹調的食品（餅乾、罐頭、能量棒）
- 急救包：OK 繃、紗布、優碘、彈性繃帶、體溫計
- 常備藥品：個人慢性病藥（至少 3 天份）、止痛藥、腸胃藥
- 手電筒 + 備用電池（或充電式）
- 行動電源（已充飽電）
- 收音機（電池式）
- 身分證件影本（放防水袋）
- 現金（災難時 ATM 可能無法使用）
- 保暖衣物與毛毯
- 口哨（被困時求救用）
- 安全口罩（N95 或以上）

## 三、緊急疏散注意事項
- 收到疏散通知，立即行動，不要猶豫或等待。
- 帶走緊急包，其他財物可事後再處理。
- 告知家人目的地與路線。
- 協助需要幫助的鄰居（老人、身障者、獨居者）。
- 遵守疏散路線指示，不要走捷徑（可能有危險）。

## 四、心理健康
- 災難後出現恐懼、焦慮、睡眠障礙是正常反應。
- 與家人、朋友保持聯繫，互相支持。
- 兒童需要特別安慰與解釋，避免讓他們看到過多災難影像。
- 需要心理支援可撥打：
  - 衛福部安心專線：1925（24 小時）
  - 張老師服務：1980
  - 生命線：1995
