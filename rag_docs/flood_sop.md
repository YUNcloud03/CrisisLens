# 淹水災害應變標準作業程序（SOP）

## 一、災前預防
- 收聽中央氣象署颱風與豪雨特報，關注淹水潛勢圖。
- 備妥緊急物資：飲用水（每人每天 3 公升）、乾糧、手電筒、備用電池、藥品、身分證件影本。
- 低窪地區住戶提前準備沙袋，阻隔水流進屋。
- 將車輛移至高處停放，避免泡水損失。
- 確認最近的避難所位置（可至縣市政府網站查詢）。

## 二、災中應變
### 室內
- 立即關閉家中電源總開關，避免漏電危險。
- 將重要文件、藥品、電子設備移至高處（二樓以上）。
- 避免使用廁所，以防污水逆流。
- 收聽廣播或政府通報，準備隨時疏散。

### 室外
- 嚴禁涉水通行：水深超過 30 公分即有掃倒危險，超過膝蓋有生命危險。
- 避免靠近水溝、河岸、橋梁，水流湍急時有掉落危險。
- 若被困車內，水尚未進入車門時立即下車；水已進入時，等水壓平衡再開門。
- 遇到緊急狀況立即撥打 119（消防）或 112（緊急）。

## 三、疏散撤離
- 依照地方政府指示，走指定疏散路線。
- 搭乘撤離車時，攜帶緊急包與藥物。
- 通知家人、鄰居一起撤離，特別協助老人、小孩、身障者。
- 到達避難所後，向工作人員登記，接受健康檢查。

## 四、災後處理
- 返家前確認政府公告，確認建物安全才可進入。
- 穿著橡膠手套與雨靴，避免接觸污水。
- 使用漂白水（1:100）消毒受污染物品與地板。
- 檢查食物與飲用水是否遭到污染，有疑慮者丟棄。
- 注意腸道傳染病（痢疾、腸炎），保持個人衛生。
- 申請政府災害救助，聯繫縣市社會局。
